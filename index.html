CLSN Framework — 4th Redesign

Framework R4

This framework redesign combines characteristics from each of the previous three redesigns to my personal website. Being very cut and dry, I managed to keep what was absolutely necessary for the website to function on desktop and mobile devices. The framework features simple left-right splitted and centered "image/text" containers, full width text sections, and basic footer content. This is a framework that can be utilised by anyone with the basic knowledge of HTML and CSS as each piece of the framework can be edited to your liking in the stylesheet file. You also don't need to worry about trying to find the fonts and plugins as they are all in the assets directory.

Google Audit Tests

Fast. Simplistic. Modern. That's what gets these Google Audit results so high. Google Audits is a very simple tool to use to find out how well you site works on the internet. All you have to do is follow the code format and you'll get similar results. Skeptical of these scores? Well, why don't you run the audit yourself right now? Also note that PWA score is moderate as Google does not understand the mobile query styles for mobile (yikes). Also, your performance score may vary if you have a lot of images added so optimise them!
